Whois Lookup :-


A whois lookup will tell you a lot of information about who owns an internet domain . It helps to gain information regarding domain name, ownership information. IP Address, Netblock data, Domain Name Servers and other information’s. Regional Internet Registries (RIR) maintain WHOIS database.
On Linux, you can run whois lookups from the command line. We’ll walk you through it.


Terms

  1. Registry :

    A company that manages a list containing a set of domain names

  2. Registrant :

    The legal owner of the domain; it’s registered to this person.

  3. Registrar :

    A registrant uses a registrar to make his or her registration.


Information

A typical whois record will contain the following information :

  • Registrant information
  • Registrant Organization
  • Registrant Country
  • Domain name server information
  • IP Address
  • IP location
  • Autonomous System Number(ASN)
  • Domain Status
  • WHOIS history

Install

The whois command was already installed on kali linux & Ubuntu 20.04 .

for other distros you have to install it manually

  • other versions of Ubuntu :
    sudo apt-get install whois

  • On Fedora, use the command :
    sudo dnf install whois

Whois Commandline

Syntax for whois lookup is

whois {url} or {IP}

For whois lookup on google.com we will use command :

whois google.com

  • clientTransferProhibited :

  • serverDeleteProhibited :

    The domain cannot be deleted .

  • serverTransferProhibited :

    The domain cannot be transferred to another registrar .

  • serverUpdateProhibited :

    The domain cannot be updated .

DNSSEC stands for Domain Name System Security Extensions , a scheme that allows a DNS name resolver to cryptographically check that the data it received from the DNS zone is valid and hasn’t been tampered with.
